READ ME

The R code for this thesis is a modification of the R code used by Dominy et al. (2025) and Graebner et al. (2022)

Repositories used to create the code for this project:
https://github.com/graebnerc/structural-change.git
https://github.com/graebnerc/liloprojections.git
https://github.com/dominyj/EconomicPolarizationEU2025.git

To reproduce all tables and figures of the study, run the scripts stored in the  folder ..code/R in the following order:

0_value_added_data.R (gathering data for the sector value added shares)
1_data_prep.R (gathering all variables and creating the main dataset macro_world.RData)
2_data_description.R (creating Figure 3.1 and Table A5) 
3_stylized_facts.R (creating Figures 2.1, 4.5, 4.6, 4.7, 4.8)
4_a_clusters_functions.R (functions for creating clusters)
4_b_period_clusters_display.R (creating Figures 4.1, 4.3, A2, A3, A4, A5, A6) 
4_c_variable_clusters_display.R (creating Figure 4.4)
4_d_main_cluster_with_cce.R (creating Figure 4.2 and Table A4)
5_tables_summary_statistics.R (creating Tables 1.1, 3.1, 3.2, A1, A3) 
6_main_cluster_with_impulse_response.R (creating Figure A7)

All figures are stored in the folder ..code/output.
The main dataset macro_world.RData is stored in the folder ..code/data
The packages required to run the code are listed in the script ..code/packages.R
