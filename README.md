# development_models_east_asia
Code for the paper: "Beyond the Developmental State: Exploring the Variety of Development Models in East Asia"

Repositories used to create the code for this project:
https://github.com/dominyj/EconomicPolarizationEU2025.git

To reproduce the datasets, as well as the tables and figures of the study, run the script Run_Analysis.R in the main folder.

All figures are stored in the folder ../output.
The main dataset macro_world.RData is stored in the folder ../data
The packages required to run the code are listed in the script ../packages.R
